# Uses

installation

```command
$ npm install twitch-helm -g
```

basic

```command
twitch
```

options, see below Options

```command
twitch [,...options]
```

# Options

### --language

use language option can specified stream player language and default is total
stream player.

```command
twitch --language en
```

### --limit

search limit range.

```command
# search only first 20
twitch --limit 20
```
